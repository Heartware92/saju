## 🗓 추후 구현 리스트 (deferred)

### Gemini 5xx 안정성 (유료 결제 후에도 발생)
- **현상**: 유료 티어에서도 정통사주 풀이 시 `Gemini 500: An internal error...` 가 사용자 화면에 그대로 노출됨
- **원인 추정**: ① gemini-2.5-flash 모델 자체 안정성 (newer 버전이라 1.5/2.0 보다 5xx 빈발) ② 한국→미국 리전 라우팅 도중 일시 장애 ③ 긴 한국어+한자 프롬프트(5~10K 토큰) 파서 이슈 ④ 재시도 로직 없음 (1회 실패 = 100% 사용자 에러)
- **권장 수정안** (작업 시 진행):
  1. `route.ts` 에서 Claude 코드 제거 (사용자가 키 추가 안 할 예정)
  2. Gemini 5xx/429 자동 1~2회 재시도 (200ms→600ms 백오프)
  3. **OpenAI gpt-4o-mini 폴백 추가** — `.env.local` 에 `OPENAI_API_KEY` 이미 존재. Gemini 모든 재시도 실패 시 OpenAI 로 자동 전환
  4. 사용자 화면에는 친화 메시지 ("AI 서비스가 일시적으로 응답하지 않아요. 잠시 후 다시 시도해주세요.") + 재시도 버튼만 노출. raw 영문 에러 메시지 노출 금지
- **우선순위**: 🟡 — 자주 발생하지는 않으나 발생 시 사용자 경험 직격탄
